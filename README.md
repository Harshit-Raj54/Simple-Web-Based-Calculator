# Simple Web-Based Calculator

A **web-based calculator** built using **HTML, CSS, and JavaScript** that performs basic arithmetic operations including **addition (+), subtraction (-), multiplication (×), and division (÷)**. The calculator handles invalid inputs and provides a user-friendly interface.

---

## Features
- Perform basic arithmetic operations: **+**, **-**, **×**, **÷**.
- Handles **decimal numbers** and invalid inputs gracefully.
- Clean, responsive, and user-friendly interface.
- Implemented **exception handling** to avoid crashes.

---

## Technologies Used
- **HTML & CSS**: For structure and styling of the calculator interface.
- **JavaScript**: Handles user input, calculation logic, and exception handling.

---
